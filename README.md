<p align="center">Api com cadastro de produtos e venda.</p>

## Technologies

This project was developed with the following technologies:

- [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
- [Prisma]
- [Postgres]
- [VS Code][vc]

## Installation

```bash
$ npm install
```

```bash
$ cd docker
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
